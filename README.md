# LavaPlatformer

Welcome on LavaPlatformer!
Jump on the platforms by using `a` and `d` keys and get the coins. Attention! There are also transpatent platforms. With the coins you can buy more skins and unlock jobs. There are 3 different jobs, complete them and you will get as reward diamonds. With the diamonds you can buy chests. Earn key to revive after game over!
